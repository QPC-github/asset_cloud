= AssetCloud

An abstraction layer around arbitrary and diverse asset stores.

== Installation

=== as a Gem

    gem install Shopify-asset_cloud -s http://gems.github.com

=== as a Rails plugin

    script/plugin install git://github.com/Shopify/asset_cloud.git
